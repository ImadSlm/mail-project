---- PROJET MAIL RUST ----

Création d'un client mail pour envoyer et recevoir du courriel

terminal (avec concurrently) :

```console
cd frontend
npm start
```
 terminal :
 
 ```console
cd mail_client
cargo run
cd ../frontend
npm run dev
```

créer un fichier .env dans mail-project/mail_client avec :

```console
EMAIL_ADDRESS=*adresse mail*
EMAIL_PASSWORD=*mdp*
```
